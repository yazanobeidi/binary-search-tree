# binary-search-tree
C++ Binary Search Tree Implementation

	Files:
		- bst.hpp - BST declarations
		- bst.cpp - BST implementation
		- main.cpp - main, runs BST Demo
		- README.txt
		- LICENCE.txt


	Instructions:
		
		1. Navigate to BST directory (cd bst)
		2. Compile source files (g++ *.cpp -o bst) ... ignore warnings
		3. Run (./bst)
